Recommended Navigation Flows

Customer Journey

  1. Browse Products (index.html)
  2. Login (login.html) or Register (register.html)
  3. Customer Home (userHome.html)
  4. View Cart (cartDetails.html)
  5. Make Payment (payment.html)
  6. View Orders (orderDetails.html)
  7. View Profile (userProfile.html)

Admin Journey

  1. Admin Login (login.html)
  2. Admin Dashboard (adminHome.html)
  3. Manage Stock (adminStock.html)
  4. Add Product (addProduct.html)
  5. Update Product (updateProductById.html)
  6. Ship Orders (unshippedItems.html)
  7. View Shipped (shippedItems.html)
